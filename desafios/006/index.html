Tags em HTML

Tags em HTML são elementos que estruturam e exibem o conteúdo em uma página da web.
Passe o mause por cima dos textos para descobrir as tags dos determinados serviços:

  • Paragrafos
  • Imagens
  • Título
  • HyperLinks
  • Negrito
  • Italico
  • Quebra de linha
  • Lista (ordenadas)
  • Listas(desordenadas)
  • Texto Deletado
